# Snakes Game

### [Game Link](https://doublesnakes.netlify.app/)

## Descirption

A game that builds on the classic game that uses a backend to allow a multi-player version of the game where the players play from separate computers.

### Installing

```
npm i
```

### Running App

```
npm start
```
